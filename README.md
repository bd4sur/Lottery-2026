## 1. 活动背景与奖品

BD4SUR手工打造了一个“卡片电子鹦鹉”，这就是本次抽奖的奖品。它是一个明信片大小的手持设备，带有键盘和屏幕。主要包括两大功能：

- 自制大语言模型“Nano”：168M参数的语言模型，具备基本的语言能力，由BD4SUR从头开始预训练和微调，仅用于技术验证。设备还搭载了Qwen3-0.6B等开源模型，但限于1GB的内存大小，无法运行。如果升级内存，或许可以运行。
- 带重力感应的天象仪“玲珑”：依据星历算法计算太阳、月亮和其他天体在天空中的位置，并渲染为逼真的视觉效果。设备带有IMU，可以根据设备的姿态自动调整观察视角。

限于技能和时间，设备的完成度不是很高，尤其没有PCB和外壳。因此，奖品的主要价值，在于那块全志A733的开发板。奖品中各部件的价格大概是：开发板（[Radxa Cubie A7Z](https://docs.radxa.com/cubie/a7z) 1GB RAM）120元左右，IMU大概40元，屏幕键盘各20元左右。

## 3. 开奖

**3.1 抽奖者编号办法**

凡获得抽奖资格的报名者，将在第0节的表格中公示，并赋予编号。编号从0开始递增。编号的依据是提交报名Issue的时间顺序，时间早者，编号小。

**3.2 开奖时间**

2026年4月4日，以当日晚间中国福利彩票3D游戏开奖结果为熵源。

**3.3 开奖办法**

- 选取[中国福利彩票3D游戏](http://www.cwl.gov.cn/fcpz/yxjs/fc3d/)在开奖日期当天的开奖结果，记为`S`，`S`是一个位于[0, 999]闭区间内的整数。
- 抽奖名单锁定后，确认获得抽奖资格的人数记为`N`，`N`不大于1000。
- 福彩最大有效开奖结果 `Smax = N * 下取整(1000 / N) - 1`。
- 若`S > Smax`，则为无效开奖，开奖向后顺延一天。
- 若`S <= Smax`，则为有效开奖，中奖者在第0节的表格中的编号记为`k`，`k = S Mod N`，其中`Mod`指计算`S`除以`N`所得的余数。
